# Capturas de pantalla

Coloca aquí las capturas del flujo principal (desde https://alquiler-justo.streamlit.app),
con **exactamente estos nombres** para que aparezcan en el README principal:

| Archivo | Qué capturar |
|---|---|
| `01-inicio.png` | Pantalla de inicio (selección de perfil inquilino/propietario) |
| `02-veredicto.png` | Resultado del análisis: veredicto + gauge + histograma |
| `03-desglose.png` | "¿Por qué este precio?" (gráfico de cascada) |
| `04-tasar.png` | Tasar mi propiedad: las 3 estrategias de precio |
| `05-mapa.png` | Mapa de precios por distrito |

**Cómo capturar:** Windows `Win + Shift + S` · Mac `Cmd + Shift + 4`. Guarda como PNG con el nombre indicado.
